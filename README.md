#### 介绍
用于促进高校学生社交以及互帮互助的微信小程序uniapp

技术栈： **uniapp+(后端)nodejs+koa+websocket** 

#### 部分页面展示
登录注册页：登录、注册

主页：首页、同学列表、最近消息、个人中心

部分页面：课程表、跑腿、文章、聊天页、疫情防控
